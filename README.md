# 🛍️ ShopSphere

A modern React-based E-commerce Web Application built using React, Context API, and React Router. ShopSphere provides a seamless online shopping experience with product browsing, authentication, wishlist, shopping cart, checkout, and order management through a clean and responsive user interface.

---

## 🔥 Features

- 🏠 Modern Responsive Home Page
- 🛍️ Product Listing & Product Details
- 🔍 Global Product Search
- 🗂️ Category Filtering
- 💰 Price Filtering
- 📊 Product Sorting
- ❤️ Wishlist Management
- 🛒 Shopping Cart
- ➕ Increase / Decrease Quantity
- 💳 Checkout System
- 📦 Order History
- 🔐 User Authentication
- 🛡️ Protected Routes
- 💾 Persistent Data using Local Storage
- 📱 Fully Responsive Design

# 🚀 How to Run the Project

### 1️⃣ Clone Repository

```bash
git clone https://github.com/durveshshinde/shopsphere-react.git

cd shopsphere-react
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run Development Server

```bash
npm run dev
```

### 4️⃣ Build for Production

```bash
npm run build
```

### 5️⃣ Open in Browser

```
http://localhost:5173/
```

# 📊 How It Works

- Browse products from the home page.
- Search products using the global search bar.
- Filter products by category and price.
- Sort products by price or name.
- View detailed product information.
- Add products to Wishlist.
- Add products to Cart.
- Manage product quantity in the cart.
- Complete checkout with shipping details.
- Place an order successfully.
- View previous orders.
- Login to access protected pages.
- Automatically persist user session and shopping data using Local Storage.
